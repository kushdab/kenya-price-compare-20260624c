# Kenya Price Compare 2026

A lightweight browser extension that automatically detects product pages on major Kenyan e-commerce platforms and provides quick links to compare prices on competing sites.

## Supported Platforms
- Jumia Kenya
- Kilimall
- Sky Garden

## How it works
1. The extension monitors the URL to see if you are on a supported store.
2. Once a product page loads, it extracts the product title.
3. A floating comparison widget appears at the bottom-right corner.
4. Click any of the platform buttons to search for that exact product on other sites.

## Installation
1. Download or clone this repository.
2. Open Chrome/Edge and navigate to `chrome://extensions`.
3. Enable "Developer mode" (toggle in the top right).
4. Click "Load unpacked" and select the directory containing these files.
